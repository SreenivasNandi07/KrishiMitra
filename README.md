# KrishiMitra
AI-based assistant for farmers that provides real-time insights on crop prices, demand trends, best market (mandi) recommendations, storage advice, and actionable guidance.

It combines:
- Historical crop data from MongoDB
- ML predictions (LSTM) for future prices
- Vector Knowledge Base (FAISS + HuggingFace embeddings)
- LLM (Ollama) for natural language explanations

## In data folder change the csv files to work on various datasets like that. 
This website is built using python and used ollama 3.1 version.
